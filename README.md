```
Converts image to color palette

Positional arguments:
  image                 Image to convert

Optional arguments:
  -h,--help             Show this help message and exit
  -t,--timing           Prints timings
  -o,--output OUTPUT    Set output name. Tries to honour set extension.
                        Example: output.png
  -c,--colors COLORS    Hexcodes in parenthesis and split by comma. Example:
                        "2E3440,3B4252,434C5E". Uses Nord color palette if not
                        set: https://www.nordtheme.com/
```
